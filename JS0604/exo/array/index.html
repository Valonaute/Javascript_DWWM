Exercices Arrays

  1. Définir un tableau Livres et y stocker deux ou trois éléments
  2. Afficher le second élément dans la console
  3. Récupérez la longueur de l'array via sa propriété length
  4. Afficher tous les éléments dans la console avec une boucle
  5. Ecrasez l'élément portant l'index 1, puis réaffichez le tableau
  6. Ajoutez un élément en début de tableau, et un autre à la fin via les methodes push() et unshift()
  7. Retirez les éléments ajoutés via shift() et pop()

Array et fonctions

  • Créez un second tableau contenant des plats avec quelques entrées
  • Affichez le contenu du tableau via une boucle automatique forEach(function(element, index){});
