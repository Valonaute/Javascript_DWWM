Fiches Personnages

Travail sur les variables

  1. Créez d'abord les variables suivantes
    • nom = string(chaîne de caractères)
    • prenom = string(chaîne de caractères)
    • job = string(chaîne de caractères)
    • age = integer(nombre entier)
    • working = boolean(opérateur booleen)
    dans le fichier js/script.js
  2. Ensuite, renseignez un autre personnage en ecrasant les variables, (y compris le boléen)
  3. Ajoutez deux ans a votre nouveau personnage (sans toucher directement aux variables)
  4. Maintenant, calculez la moyenne d'âge de vos deux personnages dans une variable que vous nommerez resultat

Option:

  1. Faites apparaître les deux fiches personnages sur cette page avec une création d'éléments
  2. Utilisez un objet plutôt que des variables libres
  3. Généralisez la création de fiches personnage avec une fonction
  4. Automatisez la création de fiches avec une boucle
